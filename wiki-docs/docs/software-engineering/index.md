# 软件工程

> 软件工程是高级工程师的基本功，体现代码质量意识、设计能力、协作能力。本模块覆盖设计原则、代码质量、Code Review、敏捷开发。

## ⭐ 面试重点速览

| 考察点 | 重要程度 | 面试频率 | 掌握目标 |
|--------|----------|----------|----------|
| SOLID 五大原则 | ⭐⭐⭐ | 极高 | 能说出每条原则并举例 |
| 代码坏味道与重构 | ⭐⭐⭐ | 高 | 能识别常见坏味道、说出重构手法 |
| Code Review 如何做 | ⭐⭐ | 高 | 知道评审要点和沟通技巧 |
| Scrum vs Kanban | ⭐⭐ | 中 | 区别和适用场景 |
