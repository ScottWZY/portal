## 每周学习建议

1. **每日任务量**：每天3-4小时，不要熬夜突击，保持节奏更重要
2. **动手实践**：理论一定要配合代码实践，看懂了不代表会写了
3. **记笔记**：好记性不如烂笔头，整理成Markdown笔记方便复习
4. **总结复盘**：每周日花1小时回顾本周内容，整理疑难问题
5. **项目驱动**：学习每个知识点后，尽量找机会做小项目验证
