# 学习导航

欢迎来到高级工程师 / AI 应用工程师面试准备 Wiki！

## 这个 Wiki 是什么？

这是一个**系统化、可执行**的面试准备知识库，帮助你在 **3-4 个月内**从 Java 中级水平提升到高级工程师 / AI 应用工程师面试水平。

## 如何使用？

### 1. 先看全局

- [**知识图谱**](./roadmap.md) — 看完整知识全貌，了解各个模块之间的关系
- [**16周学习计划**](./plan.md) — 按周拆解的可执行计划，每天学什么一目了然

### 2. 按模块逐项击破

每个模块都有独立的文档树，包含「核心原理 → 面试考点 → 实战练习」三个层次：

| 模块 | 重点 | 预计时长 |
|------|------|----------|
| [Java 高级](/java-advanced/) | JVM、并发、集合源码、设计模式 | 4 周 |
| [Spring 生态](/spring-ecosystem/) | Spring Boot 原理、Spring Cloud 微服务 | 2 周 |
| [数据库](/database/) | MySQL 优化、Redis 深度 | 3 周 |
| [中间件](/middleware/) | 消息队列、分布式理论 | 2 周 |
| [高并发架构](/high-concurrency/) | 限流熔断、系统设计 | 3 周 |
| [AI 应用](/ai-application/) | RAG、Agent、LangChain | 2 周 |
| [前端](/frontend/) | Vue/React 框架、工程化、浏览器原理、性能优化、面试题 | 2-3 周 |
| [DevOps](/devops/) | Docker、CI/CD（辅助） | 穿插 |
| [面试冲刺](/interview/) | 算法、系统设计面试、行为面试 | 最后 2 周 |

### 3. 配合 code-projects 实战

每个模块学习后，在 `code-projects` 目录找到对应的代码示例和实战项目，**动手写代码比单纯看文档效率高 5 倍**。

## 学习建议

::: tip 费曼学习法
学完一个知识点后，试着用自己的话向别人讲一遍。讲不清楚的地方，就是你没理解透的地方。
:::

::: warning 不要贪多
3-4 个月时间有限，重点是**面试高频考点 + 核心原理**，而不是面面俱到。每个模块都有标注 ⭐ 的重点内容，优先掌握。
:::

::: info 输出倒逼输入
每学完一个模块，在 Wiki 中补充自己的笔记和理解。这个 Wiki 不仅是你复习的资料，也是你学习的产出。
:::
