# 移动端与跨平台

## ⭐ 面试重点速览

| 知识模块 | 重点内容 | 面试频率 |
|----------|----------|----------|
| 微信小程序 | 双线程架构、setData 原理与优化、生命周期 | 中高 |
| React Native | JS Bridge 通信、Virtual DOM→Native 组件、新旧架构对比 | 中 |
| Flutter | Skia 自绘引擎、Dart 语言、与 RN 对比 | 中 |
| Taro / uni-app | 编译时转换、多端适配、条件编译 | 中高 |
| 跨端框架对比 | 渲染方式/性能/生态/学习成本/适用场景 | 中 |

---

## 模块概述

移动端与跨平台开发是前端工程师的重要技能延伸。随着小程序生态的繁荣和跨端框架的成熟，高级前端工程师需要了解不同平台的技术原理和选型策略，以应对多样化的业务需求。

::: tip 适用人群
准备中大厂面试的前端工程师，特别是涉及小程序、跨端开发岗位的候选人。
:::

## 学习建议

1. 优先掌握微信小程序核心原理（双线程架构 + setData 优化），这是国内面试最常见的移动端考点
2. 了解 React Native 和 Flutter 的基本原理和区别，能说清楚选型理由
3. Taro/uni-app 关注编译时转换思路，理解多端适配的本质

::: danger 容易翻车的点
- 不知道微信小程序的双线程架构设计
- 分不清 React Native 和 Flutter 的渲染方式区别
- 认为 Taro 是运行时跨端（实际是编译时）
:::
