## 二、量化技术

量化通过降低模型精度来减少显存占用和提升推理速度：

| 技术 | 精度 | 显存节省 | 速度提升 | 质量损失 |
|------|------|----------|----------|----------|
| **FP16** | 半精度 | ~50% | ~1.5x | 极小 |
| **INT8** | 8bit | ~75% | ~2x | 很小 |
| **INT4 (AWQ/GPTQ)** | 4bit | ~87% | ~3x | 可接受 |
| **GGUF (Q4_K_M)** | 混合精度 | ~80% | ~2.5x | 较小 |

## 四、API 网关设计

```mermaid
flowchart LR
    Client["客户端"] --> GW["API 网关"]
    GW --> Auth["认证/鉴权"]
    GW --> Rate["速率限制"]
    GW --> Router["模型路由"]
    Router --> M1["vLLM 实例 1<br/>（主模型）"]
    Router --> M2["vLLM 实例 2<br/>（备用模型）"]
    Router --> M3["云 API<br/>（降级兜底）"]

    GW --> Monitor["监控<br/>（Prometheus）"]
```

**网关核心功能：**
- **认证鉴权**：API Key 验证、权限控制
- **速率限制**：按用户/应用限制 QPS
- **模型路由**：根据请求类型路由到不同模型
- **降级策略**：主模型不可用时切换到备用
- **监控告警**：延迟、错误率、Token 消耗
