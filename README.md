# 高级工程师面试准备项目

## 项目结构

```
.
├── wiki-docs/          # VitePress 知识库（面试准备 Wiki）
│   └── docs/           # 文档内容
│       ├── guide/       # 学习导航、知识图谱、16周计划
│       ├── java-advanced/    # Java 高级（JVM/并发/集合/设计模式）
│       ├── spring-ecosystem/ # Spring 生态
│       ├── database/         # 数据库（MySQL/Redis）
│       ├── middleware/       # 中间件（消息队列/分布式系统）
│       ├── high-concurrency/ # 高并发架构
│       ├── ai-application/   # AI 应用（RAG/Agent/LangChain）
│       ├── frontend/         # 前端（Vue3）
│       ├── devops/           # DevOps（Docker/CI/CD）
│       └── interview/        # 面试冲刺
│
└── code-projects/      # 代码示例和实战项目
    ├── java-core/           # Java 核心示例
    ├── spring-projects/     # Spring 实战
    ├── middleware-practice/ # 中间件实践
    ├── high-concurrency-projects/ # 高并发实战
    ├── ai-projects/         # AI 应用实战
    └── fullstack-projects/  # 全栈项目
```

## 快速开始

```bash
# 进入 Wiki 目录
cd wiki-docs

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
