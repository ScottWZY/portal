# Know Share

知识分享站点，涵盖 AI 应用、算法专题、大数据、计算机网络、数据库、DevOps 等技术领域。

---

## 项目简介

本项目是一个面向开发者的技术知识库，内容覆盖：

- **AI 应用**：大模型应用开发、Agent、RAG、Prompt Engineering、LangChain、MCP 等
- **算法专题**：经典算法、数据结构、深度学习算法、机器学习算法、工程算法等
- **大数据**：Hadoop 生态、Spark、Flink 等
- **计算机网络**：TCP/UDP、HTTP/HTTPS、WebSocket、CDN 等
- **数据库**：MySQL、Redis、MongoDB、Elasticsearch、Oracle、国产数据库等
- **DevOps**：Docker、Kubernetes、CI/CD、Linux 常用命令等

---

## 快速开始

### 克隆仓库

```bash
git clone https://gitee.com/wzy_2022/know-share.git
```

### 切换分支

```bash
cd know-share
git checkout develop
```

### 拉取最新代码

```bash
git pull origin develop
```
