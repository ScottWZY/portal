й•ҝеәҰеӯ—ж®өзҡ„еҗ«д№ү

UDP й•ҝеәҰеӯ—ж®өеҢ…еҗ« UDP йҰ–йғЁ + ж•°жҚ®зҡ„й•ҝеәҰпјҢе…¶жңҖеӨ§еҖјдёә 65535пјҲ16 дҪҚдёә 2^16-1пјүпјҢеҶҚеҮҸеҺ» 8 еӯ—иҠӮйҰ–йғЁпјҢUDP еҚ•дёӘжҠҘж–ҮжңҖеӨ§ж•°жҚ®дёә 65507 еӯ—иҠӮгҖӮ

дәҢгҖҒTCP vs UDP ж·ұеәҰеҜ№жҜ” вҖӢ

иҝҷжҳҜйқўиҜ•дёӯеҮ д№Һеҝ…й—®зҡ„йўҳзӣ®пјҢдёҚиғҪеҸӘиғҢиЎЁж јпјҢиҰҒзҗҶи§Ји®ҫи®Ўе“ІеӯҰгҖӮ

еҜ№жҜ”з»ҙеәҰTCPUDP
жҳҜеҗҰйқўеҗ‘иҝһжҺҘжҳҜпјҢеҝ…йЎ»е…Ҳе»әз«ӢиҝһжҺҘеҗҰпјҢзӣҙжҺҘеҸ‘йҖҒж•°жҚ®
еҸҜйқ жҖ§дҝқиҜҒж•°жҚ®дёҚдёўгҖҒдёҚйҮҚгҖҒдёҚд№ұеәҸдёҚдҝқиҜҒпјҢеҸҜиғҪдёўгҖҒеҸҜиғҪд№ұеәҸ
дј иҫ“ж–№ејҸеӯ—иҠӮжөҒпјҢж— ж¶ҲжҒҜиҫ№з•Ңйқўеҗ‘жҠҘж–ҮпјҢдҝқз•ҷж¶ҲжҒҜиҫ№з•Ң
йҰ–йғЁејҖй”Җ20~60 еӯ—иҠӮ8 еӯ—иҠӮ
жөҒйҮҸжҺ§еҲ¶жңүпјҲж»‘еҠЁзӘ—еҸЈпјүж— 
жӢҘеЎһжҺ§еҲ¶жңүпјҲж…ўеҗҜеҠЁгҖҒжӢҘеЎһйҒҝе…Қзӯүпјүж— 
дј иҫ“йҖҹеәҰзӣёеҜ№ж…ўзӣёеҜ№еҝ«пјҲдҪҺе»¶иҝҹпјү
ж”ҜжҢҒжЁЎејҸдёҖеҜ№дёҖдёҖеҜ№дёҖгҖҒдёҖеҜ№еӨҡгҖҒеӨҡеҜ№еӨҡ
еәҸеҸ·зЎ®и®Өжңүж— 
йҖӮз”ЁеңәжҷҜж–Үд»¶дј иҫ“гҖҒHTTPгҖҒйӮ®д»¶гҖҒиҝңзЁӢзҷ»еҪ•DNSгҖҒи§Ҷйў‘зӣҙж’ӯгҖҒVoIPгҖҒжёёжҲҸ

и®ҫи®Ўе“ІеӯҰе·®ејӮ вҖӢ

TCP зҡ„и®ҫи®Ўе“ІеӯҰжҳҜеҸҜйқ дј иҫ“пјҢйҖӮеҗҲеҜ№ж•°жҚ®е®Ңж•ҙжҖ§иҰҒжұӮй«ҳзҡ„еңәжҷҜгҖӮUDP зҡ„и®ҫи®Ўе“ІеӯҰжҳҜиҪ»йҮҸеҝ«йҖҹпјҢйҖӮеҗҲеҜ№е®һж—¶жҖ§иҰҒжұӮй«ҳгҖҒиғҪе®№еҝҚе°‘йҮҸдёўеҢ…зҡ„еңәжҷҜгҖӮ

зҗҶи§ЈжҖқи·Ҝ

TCP е°ұеғҸдёҖдёӘиҙҹиҙЈд»»зҡ„еҝ«йҖ’е‘ҳпјҢжҜҸдёӘеҢ…иЈ№иҰҒзӯҫж”¶гҖҒиҰҒеӣһжү§пјҢдёўдәҶиҰҒйҮҚеҸ‘пјҢйЎәеәҸд№ұдәҶиҰҒйҮҚжҺ’гҖӮUDP е°ұеғҸдёҖдёӘдёҚз®ЎдёҚйЎҫзҡ„еҝ«йҖ’е‘ҳпјҢеҢ…иЈ№жү”еҲ°й—ЁеҸЈе°ұиө°пјҢдёўдәҶдёҚиҙҹиҙЈпјҢд№ұеәҸд№ҹдёҚз®ЎпјҢдҪҶиғңеңЁеҝ«гҖӮ

UDP дёҚж”ҜжҢҒдёҖеҜ№еӨҡе’ҢеӨҡеҜ№еӨҡвҖ”вҖ”иҝҷжҳҜ TCP зҡ„еӨ©з„¶зҹӯжқҝпјҢеӣ дёә TCP еҝ…йЎ»е…Ҳе»әз«ӢиҝһжҺҘпјҢиҖҢ UDP ж— иҝһжҺҘпјҢеҗҢдёҖдёӘ socket еҸҜд»ҘеҫҖд»»ж„Ҹең°еқҖеҸ‘йҖҒж•°жҚ®гҖӮ

дёүгҖҒUDP зҡ„еә”з”ЁеңәжҷҜ вҖӢ

3.1 дёәд»Җд№Ҳ UDP йҖӮеҗҲиҝҷдәӣеңәжҷҜпјҹ вҖӢ

UDP дёўеҢ…дёҚйҮҚдј пјҢзңҒеҺ»дәҶ TCP зҡ„йҮҚдј зӯүеҫ…ж—¶й—ҙпјҢе»¶иҝҹжӣҙдҪҺгҖӮд»ҘдёӢеңәжҷҜж•°жҚ®е®һж—¶жҖ§жҜ”е®Ңж•ҙжҖ§жӣҙйҮҚиҰҒпјҡ

еңәжҷҜдёәд»Җд№Ҳз”Ё UDP
DNS жҹҘиҜўеҚ•ж¬ЎиҜ·жұӮ-е“Қеә”пјҢжҠҘж–Үе°ҸпјҢйҮҚдј ејҖй”ҖеӨ§пјҢUDP дёўеҢ…йҮҚиҜ•еҚіеҸҜ
и§Ҷйў‘зӣҙж’ӯ/дјҡи®®е®һж—¶жҖ§дјҳе…ҲпјҢдёўеҮ её§ж— жүҖи°“пјҢTCP йҮҚдј дјҡи®©з”»йқўеҚЎйЎҝ
еңЁзәҝжёёжҲҸе»¶иҝҹж•Ҹж„ҹпјҢдёўеҢ…дәҶе°ұиҝҮеҺ»дәҶпјҢйҮҚдј зҡ„ж•°жҚ®е·Із»ҸиҝҮжңҹдәҶ
VoIP иҜӯйҹійҖҡиҜқе®һж—¶йҹійў‘пјҢе»¶иҝҹжҜ”дёўеҢ…еҪұе“ҚжӣҙеӨ§
IoT и®ҫеӨҮдёҠжҠҘи®ҫеӨҮиө„жәҗжңүйҷҗпјҢTCP ејҖй”ҖеӨ§пјҢжҠҘж–Үе°ҸпјҢйҖӮеҗҲ UDP
е№ҝж’ӯ/з»„ж’ӯTCP еҸӘиғҪжҳҜзӮ№еҜ№зӮ№пјҢUDP еҸҜд»Ҙе№ҝж’ӯеҲ°еӨҡеҸ°и®ҫеӨҮ
DHCP иҺ·еҸ– IPе®ўжҲ·з«ҜиҝҳжІЎ IPпјҢж— жі•е»әз«Ӣ TCP иҝһжҺҘпјҢеҝ…йңҖ UDP

жіЁж„Ҹ

UDP дёҚдҝқиҜҒеҸҜйқ жҖ§дёҚд»ЈиЎЁеә”з”ЁеұӮдёҚиғҪеҒҡеҸҜйқ жҖ§гҖӮеҫҲеӨҡдҪҝз”Ё UDP зҡ„еҚҸи®®дјҡеңЁеә”з”ЁеұӮиҮӘе·ұе®һзҺ°еҸҜйқ жҖ§пјҢжҜ”еҰӮ QUIC еҚҸи®®еңЁ UDP д№ӢдёҠе®һзҺ°дәҶзұ»дјј TCP зҡ„еҸҜйқ дј иҫ“пјҢдҪҶйҒҝе…ҚдәҶ TCP зҡ„дёҖдәӣй—®йўҳгҖӮ

еӣӣгҖҒUDP ж ЎйӘҢе’Ң вҖӢ

UDP ж ЎйӘҢе’ҢжҳҜеҸҜйҖүзҡ„пјҲIPv4 дёӯеҸҜйҖүпјҢIPv6 дёӯеҝ…йЎ»пјүпјҢз”ЁдәҺжЈҖжҹҘж•°жҚ®еңЁдј иҫ“иҝҮзЁӢдёӯжҳҜеҗҰеҮәй”ҷгҖӮ

UDP ж ЎйӘҢе’Ңзҡ„и®Ўз®—иҢғеӣҙеҢ…жӢ¬дёүйғЁеҲҶпјҡ

  1. дјӘйҰ–йғЁпјҲPseudo HeaderпјүпјҡеҢ…еҗ«жәҗ IPгҖҒзӣ®зҡ„ IPгҖҒеҚҸи®®еҸ·гҖҒUDP й•ҝеәҰ
  2. UDP йҰ–йғЁ
  3. UDP ж•°жҚ®

дјӘйҰ–йғЁзҡ„дҪңз”Ёпјҡж ЎйӘҢе’ҢдёҚд»…иҰҶзӣ– UDP ж•°жҚ®жң¬иә«пјҢиҝҳиҰҶзӣ–дәҶ IP ең°еқҖдҝЎжҒҜпјҢеҰӮжһң IP ең°еқҖеңЁдј иҫ“иҝҮзЁӢдёӯиў«зҜЎж”№пјҢж ЎйӘҢе’Ңд№ҹиғҪжЈҖжөӢеҮәжқҘгҖӮиҝҷжҸҗдҫӣдәҶйўқеӨ–зҡ„е®үе…ЁдҝқжҠӨгҖӮ

еҰӮжһңж ЎйӘҢе’Ңи®Ўз®—з»“жһңжҳҜ 0пјҢеҲҷд»Ҙе…Ё 1пјҲ0xFFFFпјүеҸ‘йҖҒпјҲеӣ дёә 0 иЎЁзӨәжІЎжңүдҪҝз”Ёж ЎйӘҢе’ҢпјүгҖӮ

дә”гҖҒQUIC еҚҸи®®з®Җд»Ӣ вҖӢ

QUICпјҲQuick UDP Internet ConnectionsпјүжҳҜ Google ејҖеҸ‘зҡ„еҹәдәҺ UDP зҡ„дј иҫ“еұӮеҚҸи®®пјҢHTTP/3 е°ұе»әз«ӢеңЁ QUIC д№ӢдёҠгҖӮ

5.1 QUIC и§ЈеҶідәҶд»Җд№Ҳй—®йўҳпјҹ вҖӢ

TCP еҚҸи®®и®ҫи®ЎдәҺеҮ еҚҒе№ҙеүҚпјҢеӯҳеңЁдёҖдәӣеӣәжңүй—®йўҳпјҡ

  1. йҳҹеӨҙйҳ»еЎһпјҡTCP жҳҜеӯ—иҠӮжөҒпјҢHTTP/2 иҷҪз„¶еҒҡдәҶеӨҡи·ҜеӨҚз”ЁпјҢдҪҶеә•еұӮ TCP дёўеҢ…еҗҺдјҡйҳ»еЎһжүҖжңүжөҒпјҲTCP зә§еҲ«зҡ„йҳҹеӨҙйҳ»еЎһпјүгҖӮ
  2. жҸЎжүӢе»¶иҝҹй•ҝпјҡTCP дёүж¬ЎжҸЎжүӢ + TLS жҸЎжүӢпјҢиҮіе°‘ 2-3 дёӘ RTT жүҚиғҪејҖе§Ӣдј ж•°жҚ®гҖӮ
  3. еҚҸи®®еғөеҢ–пјҡTCP еңЁж“ҚдҪңзі»з»ҹеҶ…ж ёдёӯе®һзҺ°пјҢеҚҮзә§еӣ°йҡҫпјҢдёӯй—ҙи®ҫеӨҮпјҲйҳІзҒ«еўҷгҖҒNATпјүеҜ№ TCP йҖүйЎ№еӨ„зҗҶдёҚз»ҹдёҖгҖӮ

5.2 QUIC зҡ„ж ёеҝғзү№жҖ§ вҖӢ

зү№жҖ§иҜҙжҳҺ
0-RTT жҸЎжүӢжӣҫз»ҸиҝһжҺҘиҝҮзҡ„е®ўжҲ·з«ҜпјҢеҸҜд»Ҙ 0-RTT еҸ‘йҖҒж•°жҚ®
ж— йҳҹеӨҙйҳ»еЎһжҜҸдёӘжөҒзӢ¬з«ӢпјҢдёҖдёӘжөҒдёўеҢ…дёҚеҪұе“Қе…¶д»–жөҒ
иҝһжҺҘиҝҒз§»зҪ‘з»ңеҲҮжҚўпјҲWiFiвҶ’4Gпјүж—¶иҝһжҺҘдёҚдёӯж–ӯ
еүҚеҗ‘е®үе…ЁеҠ еҜҶжҳҜеҚҸи®®еҶ…зҪ®зҡ„пјҢдёҚжҳҜеҸҜйҖүзҡ„
з”ЁжҲ·жҖҒе®һзҺ°дёҚдҫқиө–еҶ…ж ёеҚҮзә§пјҢеҸҜд»Ҙеҝ«йҖҹиҝӯд»Ј

5.3 QUIC е’Ң TCP зҡ„еҜ№жҜ” вҖӢ

100%

QUIC = жҠҠ TCP зҡ„еҸҜйқ жҖ§ + TLS зҡ„еҠ еҜҶ + HTTP/2 зҡ„еӨҡи·ҜеӨҚз”ЁпјҢйғҪеңЁ UDP д№ӢдёҠйҮҚж–°е®һзҺ°пјҢйҒҝејҖдәҶ TCP зҡ„еӣәжңүй—®йўҳгҖӮ

е»¶дјёйҳ…иҜ»

е…ідәҺ HTTP/3 е’Ң QUIC зҡ„жӣҙеӨҡз»ҶиҠӮпјҢиҜ·еҸӮи§Ғ HTTP еҚҸи®®жј”иҝӣ з« иҠӮгҖӮ

е…ӯгҖҒдәӨеҸүе…іиҒ”еҲ°е…¶д»–жЁЎеқ— вҖӢ

  • TCP еҚҸи®®пјҡеҸӮи§Ғ TCP еҚҸи®®пјҢзҗҶи§Ј TCP е’Ң UDP еңЁеҸҜйқ жҖ§гҖҒжөҒйҮҸжҺ§еҲ¶гҖҒжӢҘеЎһжҺ§еҲ¶ж–№йқўзҡ„жң¬иҙЁеҢәеҲ«
  • HTTP/3 дёҺ QUICпјҡеҸӮи§Ғ HTTP еҚҸи®®жј”иҝӣпјҢHTTP/3 еә•еұӮдҪҝз”Ё QUIC жӣҝд»Ј TCP
  • DNS и§ЈжһҗпјҡеҸӮи§Ғ DNS и§ЈжһҗпјҢDNS й»ҳи®ӨдҪҝз”Ё UDP жҹҘиҜў
  • WebSocketпјҡеҸӮи§Ғ WebSocket еҚҸи®®пјҢWebSocket еҹәдәҺ TCPпјҢдҪҶеҝғи·іжңәеҲ¶еҸҜд»ҘеҖҹйүҙ UDP зҡ„иҪ»йҮҸжҖқжғі
  • Socket зј–зЁӢпјҡеҸӮи§Ғ Socket зј–зЁӢпјҢзҗҶи§Ј UDP Socket е’Ң TCP Socket зҡ„дҪҝз”Ёе·®ејӮ

дёғгҖҒз»Ҹе…ёй«ҳйў‘йқўиҜ•йўҳ вҖӢ

Q1пјҡTCP е’Ң UDP зҡ„еҢәеҲ«жҳҜд»Җд№Ҳпјҹеҗ„иҮӘйҖӮз”ЁеңәжҷҜпјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ

з»ҙеәҰTCPUDP
иҝһжҺҘйқўеҗ‘иҝһжҺҘж— иҝһжҺҘ
еҸҜйқ жҖ§еҸҜйқ пјҲдёҚдёўдёҚйҮҚдёҚд№ұзҡ„пјүдёҚеҸҜйқ пјҢеҸҜиғҪдёўеӨұ
жңүеәҸжҖ§дҝқиҜҒжңүеәҸдёҚдҝқиҜҒжңүеәҸ
еӨҙйғЁејҖй”Җ20~60 еӯ—иҠӮ8 еӯ—иҠӮ
жөҒйҮҸжҺ§еҲ¶/жӢҘеЎһжҺ§еҲ¶жңүж— 
йҖҹеәҰзӣёеҜ№ж…ўеҝ«пјҢе»¶иҝҹдҪҺ
йҖӮз”ЁеңәжҷҜHTTPгҖҒж–Үд»¶дј иҫ“гҖҒйӮ®д»¶DNSгҖҒи§Ҷйў‘зӣҙж’ӯгҖҒжёёжҲҸгҖҒVoIP

TCP йҖӮеҗҲеҜ№ж•°жҚ®е®Ңж•ҙжҖ§иҰҒжұӮй«ҳзҡ„еңәжҷҜпјҢUDP йҖӮеҗҲеҜ№е®һж—¶жҖ§иҰҒжұӮй«ҳгҖҒиғҪе®№еҝҚе°‘йҮҸдёўеҢ…зҡ„еңәжҷҜгҖӮ

Q2пјҡUDP жңүд»Җд№ҲдјҳзӮ№пјҹдёәд»Җд№ҲдёҚз”Ё TCP еҒҡжүҖжңүдәӢпјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ UDP зҡ„дјҳзӮ№пјҡ

  1. ж— иҝһжҺҘпјҡдёҚйңҖиҰҒе»әз«ӢиҝһжҺҘпјҢиҠӮзңҒиҝһжҺҘе»әз«Ӣж—¶й—ҙ
  2. дҪҺе»¶иҝҹпјҡжІЎжңүйҮҚдј гҖҒзЎ®и®ӨгҖҒжӢҘеЎһжҺ§еҲ¶пјҢе»¶иҝҹжӣҙдҪҺ
  3. дҪҺејҖй”ҖпјҡйҰ–йғЁеҸӘжңү 8 еӯ—иҠӮпјҢTCP жңҖе°‘ 20 еӯ—иҠӮ
  4. ж”ҜжҢҒе№ҝж’ӯе’Ңз»„ж’ӯпјҡTCP еҸӘиғҪжҳҜзӮ№еҜ№зӮ№
  5. ж— йҳҹеӨҙйҳ»еЎһпјҡжҜҸдёӘжҠҘж–ҮзӢ¬з«ӢпјҢдёҖдёӘдёўеҢ…дёҚеҪұе“Қе…¶д»–жҠҘж–Ү

дёәд»Җд№ҲдёҚз”Ё TCP еҒҡжүҖжңүдәӢпјҡ

  • е®һж—¶еңәжҷҜпјҲи§Ҷйў‘гҖҒжёёжҲҸгҖҒVoIPпјүеҜ№е»¶иҝҹзҡ„е®№еҝҚеәҰжһҒдҪҺпјҢTCP йҮҚдј дјҡеҜјиҮҙж•°жҚ®иҝҮжңҹ
  • е№ҝж’ӯ/з»„ж’ӯеңәжҷҜ TCP дёҚж”ҜжҢҒ
  • з®ҖеҚ•жҹҘиҜўеңәжҷҜпјҲDNSгҖҒDHCPпјүTCP иҝһжҺҘејҖй”ҖеӨӘеӨ§пјҢеҫ—дёҚеҒҝеӨұ

Q3пјҡUDP еҰӮдҪ•е®һзҺ°еҸҜйқ дј иҫ“пјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ UDP жң¬иә«дёҚжҸҗдҫӣеҸҜйқ жҖ§пјҢдҪҶеҸҜд»ҘеңЁеә”з”ЁеұӮиҮӘе·ұе®һзҺ°гҖӮеёёи§ҒжңәеҲ¶пјҡ

  1. еәҸеҲ—еҸ·е’ҢзЎ®и®ӨжңәеҲ¶пјҡеңЁеә”з”ЁеұӮж•°жҚ®дёӯеҠ е…Ҙ seq е’Ң ack еӯ—ж®ө
  2. и¶…ж—¶йҮҚдј пјҡеҸ‘йҖҒеҗҺеҗҜеҠЁе®ҡж—¶еҷЁпјҢи¶…ж—¶жңӘж”¶еҲ°зЎ®и®ӨеҲҷйҮҚдј 
  3. ж»‘еҠЁзӘ—еҸЈпјҡеңЁеә”з”ЁеұӮе®һзҺ°жөҒйҮҸжҺ§еҲ¶
  4. еүҚеҗ‘зә й”ҷпјҲFECпјүпјҡеҸ‘йҖҒеҶ—дҪҷж•°жҚ®пјҢжҺҘж”¶ж–№еҸҜд»ҘйҖҡиҝҮеҶ—дҪҷж•°жҚ®жҒўеӨҚдёўеҢ…пјҢдёҚйңҖиҰҒйҮҚдј 

QUIC еҚҸи®®е°ұжҳҜеҹәдәҺ UDP еңЁеә”з”ЁеұӮе®һзҺ°дәҶеҸҜйқ дј иҫ“зҡ„жңҖеҘҪдҫӢеӯҗпјҲеҸҜйқ дј иҫ“ + еҠ еҜҶ + еӨҡи·ҜеӨҚз”ЁпјүгҖӮ

Q4пјҡUDP зҡ„ж ЎйӘҢе’ҢжҳҜжҖҺд№Ҳи®Ўз®—зҡ„пјҹдёәд»Җд№ҲиҰҒи®Ўз®—дјӘйҰ–йғЁпјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ UDP ж ЎйӘҢе’Ңи®Ўз®—еҢ…жӢ¬дёүйғЁеҲҶпјҡдјӘйҰ–йғЁпјҲ12 еӯ—иҠӮпјү+ UDP йҰ–йғЁпјҲ8 еӯ—иҠӮпјү+ ж•°жҚ®гҖӮ

дјӘйҰ–йғЁеҢ…еҗ«пјҡжәҗ IP ең°еқҖгҖҒзӣ®зҡ„ IP ең°еқҖгҖҒеҚҸи®®еҸ·пјҲUDP=17пјүгҖҒUDP жҖ»й•ҝеәҰгҖӮ

дёәд»Җд№ҲиҰҒи®Ўз®—дјӘйҰ–йғЁпјҡеҰӮжһңи·Ҝз”ұиҝҮзЁӢдёӯ IP ең°еқҖиў«зҜЎж”№пјҢж ЎйӘҢе’ҢеҸҜд»ҘжЈҖжөӢеҮәжқҘпјҢжҸҗдҫӣйўқеӨ–зҡ„е®үе…ЁдҝқжҠӨгҖӮеӣ дёә IP еұӮеҸӘж ЎйӘҢ IP йҰ–йғЁпјҢдёҚж ЎйӘҢдёҠеұӮж•°жҚ®гҖӮ

IPv4 дёӯ UDP ж ЎйӘҢе’ҢжҳҜеҸҜйҖүзҡ„пјҲеҸҜд»ҘеЎ« 0 иЎЁзӨәдёҚж ЎйӘҢпјүпјҢIPv6 дёӯж ЎйӘҢе’ҢжҳҜеҝ…йЎ»зҡ„гҖӮ

Q5пјҡDNS дёәд»Җд№Ҳз”Ё UDPпјҹд»Җд№Ҳжғ…еҶөдёӢ DNS дјҡз”Ё TCPпјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ DNS й»ҳи®ӨдҪҝз”Ё UDP з«ҜеҸЈ 53 зҡ„еҺҹеӣ пјҡ

  1. DNS жҹҘиҜўйҖҡеёёжҠҘж–ҮеҫҲе°ҸпјҲдёҖиҲ¬дёҚи¶…иҝҮ 512 еӯ—иҠӮпјүпјҢUDP дёҖдёӘжҠҘж–Үе°ұиғҪе®№зәі
  2. DNS жҹҘиҜўжҳҜиҜ·жұӮ-е“Қеә”жЁЎејҸпјҢдёҖж¬ЎдёҖдёӘжқҘеӣһпјҢUDP ж— иҝһжҺҘгҖҒдҪҺејҖй”ҖйқһеёёеҗҲйҖӮ
  3. еҰӮжһңдёўеҢ…пјҢе®ўжҲ·з«ҜеҸҜд»ҘйҮҚиҜ•пјҢдёҚйңҖиҰҒ TCP зҡ„еҸҜйқ жҖ§дҝқиҜҒ

DNS дҪҝз”Ё TCP зҡ„жғ…еҶөпјҡ

  1. е“Қеә”и¶…иҝҮ 512 еӯ—иҠӮпјҡDNS жҠҘж–Үи¶…иҝҮ UDP еҚ•жҠҘж–ҮеӨ§е°ҸйҷҗеҲ¶пјҢйңҖиҰҒ TCP еҲҶзүҮдј иҫ“
  2. еҢәеҹҹдј йҖҒпјҲZone Transferпјүпјҡдё»д»Һ DNS жңҚеҠЎеҷЁд№Ӣй—ҙеҗҢжӯҘеӨ§йҮҸ DNS и®°еҪ•пјҢеҝ…йЎ»з”Ё TCP дҝқиҜҒе®Ңж•ҙжҖ§
  3. DNS over TLSпјҲDoTпјүпјҡеҠ еҜҶ DNS жҹҘиҜўпјҢеҹәдәҺ TCP + TLS
  4. DNS over HTTPSпјҲDoHпјүпјҡеҹәдәҺ HTTPS зҡ„ DNS жҹҘиҜўпјҢеә•еұӮжҳҜ TCP

Q6пјҡQUIC еҚҸи®®дёәд»Җд№ҲиҰҒеҹәдәҺ UDP иҖҢдёҚжҳҜ TCPпјҹ вҖӢ

еҸӮиҖғзӯ”жЎҲпјҡ

  1. еҚҸи®®еғөеҢ–пјҡTCP е®һзҺ°еңЁж“ҚдҪңзі»з»ҹеҶ…ж ёдёӯпјҢеҚҮзә§ TCP еҚҸи®®ж ҲйңҖиҰҒеҶ…ж ёжӣҙж–°пјҢе‘Ёжңҹй•ҝгҖҒиҰҶзӣ–ж…ўгҖӮQUIC еңЁз”ЁжҲ·жҖҒе®һзҺ°пјҢеҸҜд»Ҙеҝ«йҖҹиҝӯд»ЈгҖӮ
  2. йҳҹеӨҙйҳ»еЎһпјҡTCP жҳҜеӯ—иҠӮжөҒпјҢдёҖдёӘеҢ…дёўдәҶдјҡйҳ»еЎһеҗҺйқўжүҖжңүж•°жҚ®гҖӮQUIC д»ҘжөҒдёәеҚ•дҪҚпјҢжҜҸдёӘжөҒзӢ¬з«ӢпјҢдёҖдёӘжөҒдёўеҢ…дёҚеҪұе“Қе…¶д»–жөҒгҖӮ
  3. жҸЎжүӢдјҳеҢ–пјҡQUIC жҠҠдј иҫ“еұӮжҸЎжүӢе’ҢеҠ еҜҶжҸЎжүӢеҗҲе№¶пјҢж”ҜжҢҒ 0-RTTпјҢжҜ” TCP+TLS еҝ«еҫ—еӨҡгҖӮ
  4. дёӯй—ҙи®ҫеӨҮе…је®№пјҡеҫҲеӨҡдёӯй—ҙи®ҫеӨҮпјҲйҳІзҒ«еўҷгҖҒNATпјүеҜ№жңӘзҹҘ TCP йҖүйЎ№дјҡдёўејғпјҢUDP жҠҘж–ҮдёҚеҸ—еҪұе“ҚгҖӮ
  5. иҝһжҺҘиҝҒз§»пјҡQUIC йҖҡиҝҮиҝһжҺҘ ID ж ҮиҜҶиҝһжҺҘпјҢдёҚдҫқиө– IP ең°еқҖпјҢзҪ‘з»ңеҲҮжҚўж—¶иҝһжҺҘдёҚдёӯж–ӯгҖӮ
